# 🤖 AI Analyze Setup

CyberResume uses the Anthropic Messages API.

API Endpoint:

```
https://api.anthropic.com/v1/messages
```

To enable AI:

1. Create an Anthropic API Key.
2. Use your own backend or proxy.
3. Do NOT expose your API key in client-side JavaScript.
4. Deploy securely.
